**Netflix Movie Data Analysis Project**

This project performs an Exploratory Data Analysis (EDA) on a dataset of movies to uncover trends in genres, popularity, and viewer ratings. The analysis was conducted using Python in a Google Colab environment.

**📊 Project Overview**

The goal of this analysis is to answer key business questions regarding the content available on the platform:

**Genre Trends:** Identifying the most frequent genres.

**Popularity:** Finding the most and least popular titles.

**Ratings:** Categorizing movies based on their average vote scores.

**Historical Data:** Analyzing the distribution of movie releases over the years.

**🛠️ Tech Stack**

**Language:** Python

**Libraries:** Pandas, NumPy, Matplotlib, Seaborn

**Environment:** Google Colab

**📈 Key Insights**

Based on the notebook analysis:

**Most Frequent Genre:** Thriller is the top-occurring genre in the dataset.

**Highest Popularity:** Spider-Man: No Way Home (2021) reached the peak popularity score of 5083.954.

**Release Trends:** There has been a significant surge in the number of movies filmed and released post-2000, peaking around 2020.

**Rating Distribution:** The data was categorized into four tiers: not_popular, below_avg, average, and popular to better understand audience sentiment.
